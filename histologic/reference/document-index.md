## Naming Conventions

### Worldbuilding

- `character-name-full.md` - Complete character profile
- `character-name-concise.md` - Quick character reference

### Plots
- `plot-title-full.md` - Complete plot outline
- `plot-title-concise.md` - Quick plot summary

### Novels
- `novel-name/` - Folder for novel
- `novel-name/chapter-01.md` - Individual chapters
- `novel-name/outline.md` - Novel outline
- `novel-name/notes.md` - Writing notes
